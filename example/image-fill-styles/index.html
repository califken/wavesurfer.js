Image Fill Style

Use any image element that has been loaded onto the page as the wave background.

Image Fill Style example

Important: You need to load the image before passing it to the WaveSurfer.create method, as illustrated below.

HTML

                        <!-- HTML -->
<!-- Load wave and progress images, set the style display properties to none. -->
<img width="100" src="water2.jpg" id="wavesurfer-water" style="display:none;" />
<img width="100" src="wood.jpg"   id="wavesurfer-wood"  style="display:none;" />
<div id="waveform"></div>

JS

                        /**
 * JS
 * Pass the image element as an argument for the waveColor (and/or progressColor) parameter.
 */
wavesurfer = WaveSurfer.create({
    container: document.querySelector("#waveform"),
    waveColor: document.querySelector("#wavesurfer-water"),
    progressColor: document.querySelector("#wavesurfer-wood")
});

Mixing Image, Gradient and Color fill styles

WaveSurfer Options

waveColor - string | array | HTMLImageElement - Can be CSS color value string, an array of colors, or an HTML Image Element.

progressColor - string | array | HTMLImageElement - Can be CSS color value string, an array of colors, or an HTML Image Element.
